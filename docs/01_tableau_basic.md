# 여러 개의 데이터 결합 방법

## 1. 유니온(Union)
- 기존의 데이터 파일의 컬럼 값에 추가

## 2. 조인(Join)
- 두개의 데이터를 연결하여 하나의 dataset으로 만든다.

## 3. 관계 (Relation) -> 태블로에만 있는 기능
- 조인과 비슷하지만 조인보다는 느슨한 관계를 갖는다. 하나의 dataset이 아니다.
